支持版本：Hadoop-1.2.1 Java1.6以上 Debian7.8 Fedora20

使用方式：

1、下载此代码(install.sh);

2、下载sun jdk（建议使用1.7)

    http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html

3、下载Hadoop1.2.1二进制包;

4、将三者放在当前用户目录下(~/);

5、给install.sh加执行权限

    chmod +x install.sh

6、执行shell程序开始安装,安装完毕计算机自动重启

    ./install.sh
